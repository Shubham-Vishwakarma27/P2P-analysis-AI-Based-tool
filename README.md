P2P Audit Analytics Platform

Transforming raw SAP Procure-to-Pay (P2P) data into actionable audit insights using analytics, automation, and AI.

🚀 Overview

The P2P Audit Analytics Platform is an end-to-end solution designed to analyze SAP-style procurement data and detect fraud, control gaps, and anomalies across the Procure-to-Pay lifecycle.

This project simulates a real-world audit analytics engine used by enterprise finance teams and Big4 consulting firms.

🎯 Key Features
📂 Upload SAP-style Excel datasets (Vendor, PR, PO, GRN, Invoice, Payment)
🔗 End-to-end transaction flow mapping (PR → PO → GRN → Invoice → Payment)
🔍 50+ audit & fraud detection scenarios
⚠️ Risk scoring engine (Vendor & Transaction level)
📊 Interactive dashboard (EY-style UI)
🧠 Rule-based analytics engine (extensible to AI/ML)
📈 Drill-down audit investigation view


🔍 Fraud Detection Scenarios
